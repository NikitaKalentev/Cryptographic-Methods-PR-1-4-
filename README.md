# Cryptographic-Methods-PR-1-4

Kuznechik, Magma, AES – реализация блочных шифров на Python

Описание:
Учебная реализация трёх симметричных блочных шифров:
Магма (ГОСТ Р 34.12‑2015, блок 64 бита, ключ 256 бит)
Кузнечик (ГОСТ Р 34.12‑2015, блок 128 бит, ключ 256 бит)
AES‑128 (FIPS 197, блок 128 бит, ключ 128 бит)
Режим работы – ECB, дополнение PKCS#7. Программа позволяет шифровать и расшифровывать файлы.

Требования:
Python 3.6 или выше

Использование:
bash
python main.py -c {magma|kuznechik|aes} -k <ключ в hex> -i <входной файл> -o <выходной файл> {-e|-d}

Параметры:

-c – выбор шифра: magma, kuznechik, aes

-k – ключ в шестнадцатеричном виде (32 байта для Магмы и Кузнечика, 16 байт для AES)

-i – путь к входному файлу

-o – путь к выходному файлу

-e – шифрование

-d – расшифрование

Примечание:
Код носит учебный характер, не оптимизирован и не предназначен для использования в реальных системах защиты информации.

Структура проекта:
main.py – точка входа, обработка аргументов
utils.py – вспомогательные функции (PKCS#7, умножение в GF(2^8))
magma.py – реализация шифра Магма
kuznechik.py – реализация шифра Кузнечик
aes.py – реализация AES‑128

Источники:
ГОСТ Р 34.12‑2015
FIPS PUB 197 (AES)
